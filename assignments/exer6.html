# Week 10 In-class Exercise: Compare input methods

# Details

# Instructions

You will work in groups of up to 3 to find and compare input methods for a set of data points.

You will be comparing:

  • Number of clicks/gestures(e.g. taps, shakes) required to input data
  • How well it helps prevent errors
  • How does it provide guidance to users?
  • For each data point, you will recommend which input method you think works best.
  • Remember, we're talking about a touch-screen, mobile device.

Choose THREE of these data points to compare:

No. Data Point Data to enter
1. Province British Columbia
2. City Winnipeg
3. Phone number 343-765-4321
4a. Date November 10 1991
4b. Date February 28 2023
