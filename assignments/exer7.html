# Week 11 In-class Exercise: Icon design (Removed)

# Details

Design 3 icons for the Doors Open Ottawa event app. (Event to be explained in class)

# Requirements

  1. The icons should be legible when displayed at up to 64 pixels wide, and 70 pixels high.

  2. The icons will represent the amenities available at each location, so people can easily search/scan possible locations and determine whether they are suitable.

  3. The icons will be used in two ways in the app - within the filtering control, and displayed on the detail page for each venue.

  4. You must design icons for the following 2 amenities:

    • Family Friendly
    • Bike Racks

  5. As well as ONE (your choice) from the rest of this list:

    • Accessible
    • Shuttle
    • Free Parking
    • OC Transpo nearby
    • Paid Parking
    • Saturday
    • Sunday
    • Washrooms

# Deliverable for this exercise:

  • Your submission will consist of 3 .svg files, one for each icon you designed.
  • Include a screenshot to show all 3 icons as a set (firstNameLastName_iconSet.png).
  • Filename: firstNameLastName_iconDesign.zip

Note

  • Choose your tool (Adobe Illusrator or Xd or Figma)
  • Select a pictogram for each icon
  • Create an icon for each
  • Style your icon set
  • Produce your icon set
